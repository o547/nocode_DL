<h1>使い方</h1>
<h2>１．インストール</h2>
<p>pythonをインストール</p>
<p>pip install django torch numpy pandas</p>
<h2>２．起動</h2>
<p>run.cmdで起動</p>
<p><a href="http://127.0.0.1:8000/">http://127.0.0.1:8000/</a>を開く(初回は自動)</p>
<h2>３．アプリの使用</h2>
<p>１．指定のフォーマットに従いcsvファイルをアップロード</p>
<p>２．詳細を設定してモデルを学習</p>
<p>３．出力結果を手入力またはcsvの入力で確認する</p>
